*File descriptions*

1) Main.any 
Users can open and load this file inside the AnyBody Modeling System.

The file that contains a group of trial specific data for this jw_ngait_og3 trial.

The motion capture information file for this jw_ngait_og3 trial.

4) jw_ngait_og3_emg.csv
The EMG information file for this jw_ngait_og3 trial.

5) jw_ngait_og3_knee_forces.csv
